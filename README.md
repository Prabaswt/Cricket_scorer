# 🏏 Cricket Scorer

A simple, mobile-friendly cricket scoring app you can use right from your browser — no install needed.

## Live Demo
👉 https://prabaswt.github.io/Cricket_scorer/

## Features

- **Match Setup** — Enter both team names and set overs per innings (quick presets or any custom number)
- **Live Scoreboard** — Tracks runs, wickets, and overs in real time with a progress bar
- **Ball-by-Ball Log** — Color-coded chips for every delivery: runs, fours, sixes, wickets, wides, and no balls
- **Stats Strip** — Live count of fours, sixes, wickets, and wides at a glance
- **Celebrations** — Popup alerts for fours, sixes, and wickets
- **Two Innings** — Automatically sets the target after the 1st innings and tracks the chase
- **1st Innings Summary** — Compact score summary visible during the 2nd innings chase
- **Edit Overs Mid-Match** — Tap the ✏️ pencil to update max overs anytime during the 1st innings
- **Confirmation Dialogs** — Warns you before accidentally starting a new match or switching innings
- **Undo** — Remove the last ball entry at any time

## How to Use

1. Enter both team names and choose the number of overs
2. Tap the run buttons (0–6), Wicket, Wide, or No Ball after each delivery
3. At the end of the 1st innings, tap **Start Chase** to begin the 2nd innings
4. The app tracks the target and announces the result automatically

## Tech

Pure HTML, CSS, and JavaScript — single file, no frameworks, no dependencies.
